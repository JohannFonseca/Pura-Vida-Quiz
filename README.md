# Pura Vida Quiz

Aplicación web interactiva desarrollada con Vue 3 que permite aprender sobre Costa Rica de manera divertida mediante un sistema de preguntas con diferentes niveles de dificultad

---

## Tecnologías utilizadas

* Vue 3 (Framework principal)
* Vite (Entorno de desarrollo)
* JavaScript (Lógica)
* CSS (Estilos)

---

## Funcionalidades

* Pantalla de inicio, juego y resultados
* Sistema de preguntas tipo quiz
* Niveles de dificultad (fácil, medio, difícil)
* Temporizador dinámico
* Sistema de puntuación
* Efectos de sonido (aciertos, errores y resultado)
* Carga de datos desde archivo JSON
* Interfaz responsive

---

## Instalación y ejecución

1. Clonar el repositorio:

```bash
git clone https://github.com/johannfonseca/Pura-Vida-Quiz.git
```

2. Entrar al proyecto:

```bash
cd pura-vida-quiz
```

3. Instalar dependencias:

```bash
npm install
```

4. Ejecutar el servidor:

```bash
npm run dev
```

5. Abrir en el navegador:

```bash
http://localhost:5173/
```
